## 产品概述

地点搜索服务2.0是一类 Web API 接口服务；服务提供多种场景的地点搜索能力，包括关键字搜索、周边搜索、多边形区域搜索、ID 搜索。

目前搜索是不支持返回全量数据的，同请求参数翻页查询最多支持获取200条数据
